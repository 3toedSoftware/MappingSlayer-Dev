1. Getting Started

What is Mapping Slayer?

Mapping Slayer is a professional web-based tool for creating detailed sign location maps on PDF documents. Designed for signage professionals, project managers, and contractors, it enables precise annotation of architectural drawings, floor plans, and site maps with numbered location markers and detailed information.

System Requirements

Supported Browsers

  • Chrome 90+ - Recommended for best performance
  • Firefox 88+ - Fully supported
  • Safari 14+ - Fully supported
  • Edge 90+ - Fully supported

Technical Requirements

  • JavaScript must be enabled
  • Minimum 4GB RAM (8GB recommended for large projects)
  • Stable internet connection for initial loading
  • Screen resolution of 1280x720 or higher

First Time Setup

  1. Open Mapping Slayer in your web browser
  2. Load a PDF by clicking in the upload area or using LOAD
  3. Configure marker types for your project needs
  4. Enable tooltips via HELP ‚Üí TOOL TIPS for contextual help

Supported File Types

Format Extension Purpose
PDF Documents .pdf Base maps for annotation (architectural drawings, floor plans, site maps)
Slayer Projects .slayer Complete project files including PDF and all annotations

3. Working with Marker Types

Marker types categorize your location dots, each with unique colors and properties.

Default Marker Types

Mapping Slayer starts with three default types:

  • INT - Interior locations (green)
  • EXT - Exterior locations (red)
  • HCP - Handicap/Accessible locations (blue)

Creating Custom Marker Types

  1. Click the + button in the marker types section
  2. Enter a code (1-4 characters, e.g., "DOOR", "ELEV")
  3. Enter a descriptive name
  4. The new type appears with an auto-assigned color

Customizing Marker Types

Edit Names

Click on any marker type name to edit it directly. Press Enter to save.

Change Colors

  1. Click the colored circle next to a marker type
  2. Use the color picker to select dot color
  3. Click the text color circle to change text color
  4. Colors update immediately on all dots

Add Design References

  1. Click the + in the design reference box
  2. Select an image file (PNG, JPG, SVG)
  3. Image appears as reference for that type
  4. Click the image to update or remove

Managing Marker Types

Delete a Type

Right-click on the marker type name and confirm deletion. Warning: This removes all dots of that type!

Import/Export Types

  • EXP - Save marker types as JSON file for reuse
  • IMP - Load previously saved marker types

Filter Display

Use the checkboxes next to each type to show/hide those dots on the map.

4. Placing and Managing Location Dots

Adding Dots Manually

  1. Select a marker type from the left panel (it highlights in orange)
  2. Click anywhere on the PDF to place a dot
  3. The dot appears with an auto-generated location number
  4. If SHOW MSG1 is active, dots display message fields
Hold Shift while clicking to place multiple dots quickly without deselecting the tool.

Selecting Dots

Single Selection

  • Click on any dot to select it
  • Selected dots show an orange outline
  • Click again to deselect

Multiple Selection

  • Box select: Click and drag to draw selection rectangle
  • Shift-click: Add individual dots to selection
  • Ctrl/Cmd+A: Select all dots on current page
  • Find All: Select all dots matching search criteria

Moving and Copying Dots

Move Dots

  1. Select one or more dots
  2. Drag any selected dot to move all selected dots
  3. Dots maintain relative positions when moving groups

Copy and Paste

  1. Select dots to copy
  2. Press Ctrl+C to copy
  3. Press Ctrl+V to paste at cursor position
  4. Pasted dots retain all properties except location numbers

Deleting Dots

  • Single dot: Select and press Delete key
  • Multiple dots: Select all desired dots and press Delete
  • Via edit modal: Right-click and use DELETE button

5. Editing Locations

Quick Edit in List

The location list provides direct editing of messages:

  • Click in any MESSAGE 1 field to edit
  • Click in MESSAGE 2 field (when visible) to edit
  • Changes save automatically when you click elsewhere
  • Tab key moves between fields

Edit Location Modal

Right-click any dot to open the detailed edit modal:

Available Fields

  • Marker Type - Change the dot's category
  • Location Number - Modify the identifying number
  • Message fields - Edit associated text (based on project settings)
  • Notes - Add detailed information
  • Installed checkbox - Mark as complete
  • Flag checkboxes - Apply custom flags

Modal Actions

  • UPDATE - Save all changes
  • DELETE - Remove this location
  • CANCEL - Close without saving
  • SIGN PREVIEW - View the sign template for this location
  • REF IMG - View and manage reference image for this location
Use the SIGN PREVIEW and REF IMG toggle buttons at the top of the Edit Location modal for quick access to sign templates and reference images without closing the edit window.

Bulk Editing

Select multiple dots and right-click to edit them together:

  • Change marker type for all selected
  • Apply the same message to all
  • Mark all as installed
  • Set flags for entire group
  • Delete all selected at once

Custom Flags

Configure up to 4 custom flags for special categorization:

  1. Right-click on "MARKER TYPES" header text
  2. Name each flag position
  3. Choose symbols or upload custom icons
  4. Apply flags to dots via edit modal

6. Navigation and Search

Page Navigation

  • Use < and > buttons to change pages
  • Page counter shows current page and total
  • Keyboard: Page Up / Page Down
  • Location list shows page numbers for each dot

Zoom and Pan

Zooming

  • Mouse wheel: Scroll to zoom in/out at cursor position
  • Double-click dot in list: Zoom to that location
  • Middle-click dot in list: Center on that location

Panning

  • Middle-click + drag: Pan around the map
  • Arrow keys: Nudge view in small increments

Search Features

Find Locations

  1. Enter text in the Find field
  2. Press Enter or click FIND ALL
  3. All matching locations are selected
  4. Search includes location numbers and all message fields

Replace Text

  1. Search for text to replace
  2. Enter replacement text in Replace field
  3. Click REPLACE
  4. Only affects selected locations

Crop Mode

Focus on specific areas of large drawings:

  1. Click the crop button (icon changes to indicate active)
  2. Click and drag to define crop area
  3. Only the selected area remains visible
  4. Click crop button again to restore full view

7. Automap Features

Automap automatically places dots at text locations in your PDF. This feature works best with searchable PDFs containing real text (not scanned images).

Using Automap

  1. Enter text pattern in "Text to find" field (e.g., "DOOR", "RM", "ELEV")
  2. Select the marker type for new dots
  3. Click AUTOMAP IT!
  4. Progress window shows dots being placed
  5. Review results and adjust as needed

Automap Patterns

Simple Text

Enter exact text to find: DOOR, EXIT, ROOM

With Numbers

Include numbers for specific matches: DOOR 101, RM 2

Partial Matches

Automap finds text containing your pattern: searching RM finds "RM 101", "STORM", "PLATFORM"

Automap only works with searchable PDFs. For scanned documents, use OCR scraping or manual placement.

8. Text Scraping and OCR

Quickly capture text from your PDF to use as location messages.

Standard Text Scraping

For searchable PDFs with selectable text:

  1. Hold Shift + right-click and drag
  2. Yellow box appears around selected area
  3. Release to capture text
  4. Text automatically fills MESSAGE 1 of selected dots

OCR Scraping

For scanned PDFs or images without selectable text:

  1. Hold Shift + Ctrl + right-click and drag
  2. Purple box indicates OCR mode
  3. Processing extracts text from image
  4. Results populate selected dots' messages
Select multiple dots before scraping to apply the same text to all selected locations.

9. Saving and Loading Projects

Saving Your Work

Save Project

  1. Click SAVE to save with current name
  2. First save prompts for filename
  3. Creates .slayer file containing everything

Save As

  1. Click SAVE AS to save with new name
  2. Enter desired filename
  3. Original file remains unchanged

Loading Files

Load PDF

  1. Click LOAD or click upload area
  2. Select a PDF file
  3. Starts fresh project with default marker types

Load Project

  1. Click LOAD
  2. Select a .slayer file
  3. Restores complete project state
Save frequently and use descriptive names like "BuildingA_Floor2_Rev3.slayer" to track versions.

10. Export Options

PDF Export

Generate annotated PDFs with your location markers:

  1. Click CREATE PDF
  2. Choose export format:
    • Current Map - With Detail Pages: Current page plus detail sheets
    • Current Map Only: Just the current page with dots
    • All Maps Only: All pages with dots, no details
  3. PDF downloads with annotations embedded

Message Schedule (Excel)

Export location data for spreadsheet management:

  1. Click CREATE MESSAGE SCHEDULE
  2. Excel file downloads with all location data
  3. Edit messages, notes, and properties in Excel
  4. Use UPDATE FROM MESSAGE SCHEDULE to reimport changes

HTML Report

Create web-based project documentation:

  1. Click EXPORT HTML
  2. Generates interactive HTML report
  3. Includes maps, location lists, and summaries
  4. Share via email or web hosting

11. Sign Templates and Preview

Mapping Slayer includes powerful sign template features with Grade 2 Braille translation support, allowing you to preview signs and generate professional thumbnail sheets.

Sign Preview

View sign templates directly from the Edit Location modal:

  1. Right-click any dot to open the Edit Location modal
  2. Click SIGN PREVIEW at the top of the modal
  3. The sign template for that marker type displays in a preview window
  4. Preview shows the actual sign design with all text and braille
  5. Click anywhere outside to close the preview
The SIGN PREVIEW button stays in the Edit Location modal, so you can preview the sign while editing location details.

Grade 2 Braille Translation

Mapping Slayer uses LibLouis for authentic Grade 2 Braille translation on all sign templates:

Braille Features

  • Automatic Translation - Text converts to Grade 2 Braille automatically
  • Unicode Support - Proper Unicode braille characters for display
  • US English Standard - Follows en-us-g2.ctb standards
  • Custom Font - Uses embedded Braille font for PDF exports

Create Thumbnails

Generate professional PDF sheets displaying all your sign designs:

  1. Click CREATE THUMBNAILS in the right panel
  2. Select which marker types to include in the thumbnail sheet
  3. Choose your page size:
    • Letter (8.5" √ó 11")
    • Legal (8.5" √ó 14")
    • Tabloid (11" √ó 17")
    • Custom dimensions
  4. Set columns per page (1-6 columns)
  5. Adjust sign scale percentage (50-200%)
  6. Choose margin size
  7. Click GENERATE
  8. PDF downloads with all sign thumbnails in a grid layout

Thumbnail Options

Layout Control

  • Columns: More columns = smaller signs, fits more per page
  • Scale: Adjust sign size without changing column count
  • Margins: Small, Medium, or Large spacing around signs

Best Practices

  • Use 2-3 columns for detailed review of large signs
  • Use 4-6 columns for compact reference sheets
  • Tabloid paper works best for multiple columns
  • Keep scale at 100% for actual-size preview
Thumbnail PDFs include location information and are perfect for client presentations, fabrication references, and project documentation.

Reference Images

Add photos to individual locations for reference during installation or fabrication. These photos are automatically included in PDF exports.

Adding Reference Images

  1. Right-click any dot to open the Edit Location modal
  2. Click REF IMG at the top of the modal
  3. Click the + button or upload area to add a photo
  4. Photos are automatically compressed and stored with the location

Exporting All Photos

Export all reference images from your project as a ZIP file:

  1. Click EXPORT PHOTOS in the footer controls
  2. Choose where to save the ZIP file
  3. Photos are named with full metadata for easy organization
Photo Filename Format:
Photos are exported as: ProjectName-MarkerID-MarkerName-page#(LocationNumber).jpg
Example: EXAMPLE_MAPS-ID.1-InteriorDoor-page1(0001).jpg
Reference images automatically appear in CREATE PDF exports instead of generic design references, providing accurate field documentation.

13. Special Features

Renumbering

Reorganize location numbers for professional presentation:

Renumber Options

  • Current Page Only: Renumber by position (top-to-bottom, left-to-right)
  • Current Page by Marker Type: Separate numbering per type
  • All Pages: Continuous numbering across document
  • All Pages by Marker Type: Type-specific numbering throughout

DOTCAM Mode

Capture reference images directly when placing new locations:

  1. Click DOTCAM to activate
  2. Place a new dot - camera opens automatically
  3. Take photo or select from device
  4. Photo is saved as the location's reference image
  5. View it later using the REF IMG button

Installation Tracking

Monitor project completion:

  • Mark dots as "Installed" via edit modal
  • Use INST ONLY to show only installed items
  • Legend displays installation counts
  • Installed dots show special indicator in list

Annotation Lines

Create visual connections between dots and specific points:

  1. Select a dot
  2. Hold Ctrl and drag to target location
  3. Arrow line connects dot to point
  4. Useful for indicating exact positions in complex areas

14. Tips and Best Practices

Project Organization

  • Consistent Naming: Use standard codes across projects
  • Color Coding: Assign meaningful colors (red=emergency, blue=accessible)
  • Regular Saves: Save after completing each section
  • Version Control: Keep dated backups of important projects

Efficiency Tips

  • Bulk Operations: Select multiple dots before editing
  • Keyboard Navigation: Learn shortcuts to work faster
  • Automap First: Use automatic placement, then refine manually
  • Template Types: Export and reuse marker type configurations

Quality Checks

  • Review Mode: Hide messages to see dot placement clearly
  • Legend Verification: Check counts match expectations
  • Search Duplicates: Find repeated location numbers
  • Export Preview: Generate PDF to review before sharing

15. Troubleshooting

Common Issues and Solutions

PDF Won't Load

  • Check file isn't corrupted - try opening in PDF reader
  • Ensure file size is under 100MB
  • Try a different browser if issue persists

Automap Not Finding Text

  • Verify PDF contains searchable text (not just images)
  • Try simpler search patterns
  • Check for special characters or formatting
  • Use OCR scraping for scanned documents

Dots Not Visible

  • Check marker type filters aren't hiding dots
  • Verify you're on the correct page
  • Try "SHOW ALL" instead of "INST ONLY"
  • Check if crop mode is active

Can't Select Dots

  • Ensure you're not in placement mode
  • Try clicking directly on dot center
  • Use box selection for small dots
  • Check zoom level - zoom in for precision

Export Issues

  • Allow pop-ups for PDF downloads
  • Check browser download settings
  • Ensure project has dots before exporting
  • Try different export format

Performance Optimization

  • Large Projects: Hide unnecessary marker types
  • Slow Response: Reduce number of visible dots
  • Memory Issues: Close other browser tabs
  • Refresh Page: Save project, refresh, and reload

Data Recovery

  • Browser Crash: Check Downloads folder for auto-saved files
  • Accidental Deletion: Use Undo (Ctrl+Z) immediately
  • Lost Work: Always save before closing browser
